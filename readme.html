Описание проекта "Сайт о недвижимости" Задача Разработать сайт о недвижимости с использованием WordPress, подключить тему UnderStrap, и создать функционал для управления базой данных объектов недвижимости и городов. Требования 1. Настройка сайта 1.1. Установка WordPress на бесплатном хостинге или собственном. 1.2. Установка темы UnderStrap. 1.3. Создание дочерней темы для настройки и доработки. 2. База недвижимости 2.1. Создание типа поста "Недвижимость". 2.2. Создание таксономии "Тип недвижимости" с типами: частный дом, квартира, офис и др. 2.3. Возможность добавления фотографий объекта недвижимости. 2.4. Метаполя: Площадь Стоимость Адрес Жилая площадь Этаж 2.5. Добавление 11-20 записей объектов недвижимости. 3. База Городов 3.1. Использование post_type для хранения данных о городах. 3.2. Установление связи между постами "Недвижимость" и "Города". 3.3. Добавление данных о городах: название, описание, и выбор картинки с фотографией города. 4. Главная страница 4.1. Вывод последних объектов недвижимости и городов на главной странице. 4.1.1. Пользователь может открыть страницу с объектом недвижимости и просмотреть его данные. 4.1.2. Пользователь может открыть страницу с городом и увидеть до 10 последних объектов недвижимости в этом городе. 4.2. Отображение характеристик объектов недвижимости в списке на главной. 4.3. AJAX-форма добавления объекта недвижимости с заполнением основных полей. 5. Прочее 5.1. Создание пользователя-администратора и предоставление доступа для проверки. 5.2. Создание public репозитория на GitHub и загрузка кода сайта. 5.3. Отправка данных для оценки результата. Примечание Пожалуйста, обеспечьте полный функционал и проверьте работоспособность всех элементов сайта перед предоставлением доступа и отправкой данных для оценки.
