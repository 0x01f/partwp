# Сайт о недвижимости ## Задача Разработка сайта о недвижимости с использованием WordPress, подключение темы UnderStrap и создание функционала для управления базой данных объектов недвижимости и городов. ## Требования ### 1. Настройка сайта - Установка WordPress на бесплатном хостинге или собственном. - Установка темы [UnderStrap](https://understrap.com/). - Создание дочерней темы для настройки и доработки. ### 2. База недвижимости - Создание типа поста "Недвижимость". - Создание таксономии "Тип недвижимости" (частный дом, квартира, офис и др.). - Возможность добавления фотографий объекта недвижимости. - Метаполя: Площадь, Стоимость, Адрес, Жилая площадь, Этаж. - Добавление 11-20 записей объектов недвижимости. ### 3. База Городов - Использование post_type для хранения данных о городах. - Установление связи между постами "Недвижимость" и "Города". - Добавление данных о городах: название, описание, и выбор картинки с фотографией города. ### 4. Главная страница - Вывод последних объектов недвижимости и городов на главной странице. - Пользователь может открыть страницу с объектом недвижимости и просмотреть его данные. - Пользователь может открыть страницу с городом и увидеть до 10 последних объектов недвижимости в этом городе. - Отображение характеристик объектов недвижимости в списке на главной. - AJAX-форма добавления объекта недвижимости с заполнением основных полей. ### 5. Прочее - Создание пользователя-администратора и предоставление доступа для проверки. - Создание public репозитория на [GitHub](https://github.com/) и загрузка кода сайта. - Отправка данных для оценки результата.
